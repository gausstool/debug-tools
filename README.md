# debug-tools

调试工具箱，支持文本、URL、SQL、HTTP 等常用调试工具。

## Features

### 文本相关工具

- [x] TEXT 对比
- [x] TEXT 排序
- [x] TEXT 大小
- [x] URL 解析
- [x] URL 编码
- [x] URL 解码
- [x] Base64 编码
- [x] Base64 解码

### SQL 相关工具

- [x] SQL 压缩
- [x] SQL 格式化

### 请求相关工具

- [x] CSP 解析
- [x] CSP 逆解析
- [x] HTTP 缓存解读
- [x] HTTP 跨域解读

### 其他工具

- [x] 正则表达式匹配
- [ ] 随机数生成
- [ ] 时间戳转换
- [ ] 颜色转换
- [ ] 随机端口生成
- [ ] 随机 IP 生成
- [ ] 随机头像生成
- [ ] UUID 生成
