# debug-tools

调试工具箱，支持文本、URL、SQL、HTTP 等常用调试工具。

## Features

### 文本相关工具

- [x] TEXT 排序
- [x] TEXT 大小
- [x] Text 分号分割
- [x] Text 逗号分割
- [x] Text 换行分割
- [x] SRT 转 TXT

### URL 相关工具

- [x] URL 解析
- [x] URL 编码
- [x] URL 解码
- [x] URL 参数编码
- [x] URL 参数解码

### Base64 相关工具

- [x] Base64 编码
- [x] Base64 URL Safe 编码
- [x] Base64 解码

### CSS 相关工具

- [x] CSS 转 Sass
- [x] CSS 转 Scss

### 请求相关工具

- [x] CSP 解析
- [x] CSP 逆解析
- [x] HTTP 缓存解读
- [x] HTTP 跨域解读

### SQL 相关工具

- [x] SQL 压缩
- [x] SQL 格式化

### 随机工具

- [x] 随机 UUID 生成
- [x] 随机数字生成
- [x] 随机密码生成
- [x] 随机端口生成

### 其他工具

- [x] 正则表达式匹配
- [x] Nginx 日志解析
